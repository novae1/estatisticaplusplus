1.1. Características da Linguagem Fortran

Fortran, especialmente na sua versão 90, apresenta-se como uma linguagem de programação compilada. Isso significa que o código-fonte, escrito em Fortran, é transformado pelo compilador em código de máquina, diretamente executável pelo computador. Nesta seção, exploraremos as características básicas e a estrutura de um programa em Fortran, juntamente com a compilação e execução de um arquivo fonte.

  • Estrutura de um programa: A programação em Fortran começa com um bloco program, que define o início do código executável. Todas as operações e declarações de variáveis são feitas dentro deste bloco.
  • Compilação e execução: Um programa em Fortran é transformado em código de máquina através de um compilador, como o gfortran. O arquivo resultante é um executável que pode ser diretamente executado no sistema operacional.
  • Tipagem forte: Fortran é uma linguagem fortemente tipada, o que significa que o tipo de cada variável deve ser explicitamente declarado (integer, real, complex, character, logical).
  • Operações básicas: Fortran oferece uma gama de operadores aritméticos e lógicos padrão, permitindo a realização de cálculos e manipulações lógicas dentro do programa.

Ao fim desta seção, teremos uma compreensão suficiente sobre a base da programação em Fortran, preparando o terreno para abordagens mais complexas em seções subsequentes.

1.2. Estrutura Básica de um Programa

A estrutura básica de um programa em Fortran é composta pelo bloco program, seguido por declarações de variáveis e o corpo principal do programa que contém a lógica a ser executada. O programa é finalizado com end program. Esta estrutura garante uma organização clara e a fácil localização de erros e funções dentro do código.

Exemplo de um programa simples em Fortran: 

            
                program exemplo
                    print *, 'Olá mundo!'
                end program exemplo

Este exemplo demonstra um programa que imprime "Olá mundo!" na tela, ilustrando a simplicidade da escrita de programas em Fortran e a forma como são estruturados.

1.3. Variáveis e Tipos de Dados

Fortran define variáveis dentro de tipos específicos que determinam a natureza dos dados que a variável pode armazenar. A declaração de tipos é rigorosa, suportando tipos como integer, real, complex, character, e logical, cada um com usos específicos dentro de cálculos matemáticos ou manipulações de dados.

As variáveis em Fortran são declaradas da seguinte forma, onde o tipo de dados precede o nome da variável, garantindo clareza e rigor na tipagem: 

            
                integer :: contador
                real :: distancia
                complex :: z
                character(len=1) :: letra
                logical :: valido

Essa seção introduzirá o conceito de variáveis e tipos de dados, fundamentais para a manipulação de informações e a realização de cálculos dentro de programas em Fortran.
